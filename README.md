# What-day
This application will tell you what day when you enter the day.

# What is What-day?
When you enter a day, What-day tells you what happened on that day and what is characteristic of that day.

# what kind of data
User
  * id
  * password
  * name

Necessary information
  * The month you want
  * The day you want

# dependencies
  * express
  * express-session
  * body-parser 
  * mongoose
  * passport
  * passport-local
  * bcrypt
